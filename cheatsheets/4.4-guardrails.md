# 4.4 Amazon Bedrock Guardrails — 치트시트

## TL;DR
Guardrails = 에이전트 입출력 제어 안전장치. 대회에서는 **최소한으로 설정** — 과도한 제한이 오히려 점수 손실.

## 기본 생성 패턴
```python
import boto3

bedrock = boto3.client('bedrock', region_name='ap-northeast-2')

guardrail = bedrock.create_guardrail(
    name='ai-league-guardrail',
    description='AI League 대회용 기본 가드레일',
    blockedInputMessaging='입력이 차단되었습니다.',
    blockedOutputsMessaging='출력이 차단되었습니다.',
    # 대회에서는 contentPolicyConfig를 최소화
)
guardrail_id = guardrail['guardrailId']
guardrail_version = guardrail['version']
```

## 에이전트에 연결
```python
bedrock_agent.update_agent(
    agentId=os.environ['AGENT_ID'],
    guardrailConfiguration={
        'guardrailIdentifier': guardrail_id,
        'guardrailVersion': guardrail_version,
    }
)
```

## 대회 전략
- **기본 설정만 사용**: 불필요한 토픽 차단 설정 X
- **게임 맥락 허용**: 지도 탐색, 코인 수집 관련 내용이 차단되지 않도록
- **테스트 필수**: 가드레일 적용 후 실제 게임 응답이 차단되는지 확인

## 주요 에러
| 에러 | 원인 | 해결 |
|------|------|------|
| GuardrailIntervention | 응답이 가드레일에 차단됨 | 가드레일 설정 완화 |
